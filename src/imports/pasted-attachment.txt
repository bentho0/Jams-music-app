🔄 Implement Tunemymusic Integration — Playlist Export Feature
Add a "Save to Spotify" flow using Tunemymusic as the bridge. Remove any existing Spotify OAuth playlist-saving code and replace it entirely with this implementation. Do not change any other feature, UI layout, or playlist generation logic.

What This Does
Instead of saving directly to Spotify via API, the app formats the generated playlist as plain text and redirects the user to Tunemymusic with the track list ready to transfer to Spotify, Apple Music, YouTube Music, or any platform they choose.

Step 2 — Build the Playlist Text Formatter
Create a function that converts the generated playlist object into plain text that Tunemymusic can read:
javascriptfunction formatPlaylistForTunemymusic(playlist) {
  const trackList = playlist.tracks
    .map(track => `${track.track_name} - ${track.artist}`)
    .join("\n");

  return trackList;
}
```

Output should look exactly like this:
```
God's Plan - Drake
Nights - Frank Ocean
Redbone - Childish Gambino
Location - Khalid
Pink + White - Frank Ocean
No numbering, no extra formatting, no album names. Just Track Name - Artist on each line. This is the exact format Tunemymusic accepts.

Step 3 — Build the Tunemymusic Redirect Function
javascriptfunction saveToSpotifyViaTunemymusic(playlist) {

  // Format the track list as plain text
  const trackList = formatPlaylistForTunemymusic(playlist);

  // Encode it for URL safe transfer
  const encodedTracks = encodeURIComponent(trackList);
  const encodedTitle = encodeURIComponent(playlist.playlist_title);

  // Build the Tunemymusic URL
  const tunemymusicUrl = `https://www.tunemymusic.com/transfer?source=text&playlistName=${encodedTitle}&songs=${encodedTracks}`;

  // Open in a new tab so user doesn't lose their generated playlist
  window.open(tunemymusicUrl, "_blank");
}
