Add a complete playlist sharing system to the existing playlist screen. This includes a share button, a share modal, a native mobile share flow, and a publicly accessible shareable playlist page. Do not change any existing UI, layout, or functionality — only add the new components described below.

Step 1 — Generate a Unique Shareable URL for Every Playlist
Every time a playlist is generated, create a unique ID and store the playlist data against it in Supabase:
javascript// Generate unique playlist ID on creation
const playlistId = crypto.randomUUID();

// Store playlist in Supabase
const { data, error } = await supabase
  .from("shared_playlists")
  .insert({
    id: playlistId,
    title: playlist.playlist_title,
    description: playlist.playlist_description,
    mood_tags: playlist.mood_tags,
    tracks: playlist.tracks,
    created_at: new Date().toISOString(),
  });

// Build the shareable URL
const shareUrl = `${window.location.origin}/playlist/${playlistId}`;
```

Create the following Supabase table:
```
shared_playlists
- id (text, primary key)
- title (text)
- description (text)
- mood_tags (json)
- tracks (json)
- created_at (timestamp)
Set row-level security to public read, no auth required — anyone with the link can view it.

Step 3 — Mobile: Native Share Sheet
On mobile, trigger the device's native share sheet directly:
javascriptasync function triggerNativeShare(playlist, shareUrl) {

  const shareText = buildShareText(playlist, shareUrl);

  try {
    await navigator.share({
      title: playlist.playlist_title,
      text: shareText,
      url: shareUrl
    });
  } catch (error) {
    // User cancelled or share failed — do nothing, no error shown
  }
}
```

No modal needed on mobile. The native share sheet covers WhatsApp, Instagram, Telegram, iMessage, and everything else the user has installed.
```

Step 5 — Build the Share Text
javascriptfunction buildShareText(playlist, shareUrl) {

  const topTracks = playlist.tracks
    .slice(0, 3)
    .map(t => `${t.track_name} - ${t.artist}`)
    .join("\n");

  const remaining = playlist.tracks.length - 3;

  return `🎵 ${playlist.playlist_title}

${playlist.playlist_description}

${topTracks}
... and ${remaining} more

Listen & save → ${shareUrl}

Generated by [Your App Name]`;
}
